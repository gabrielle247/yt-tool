**the_yt** is a cutting-edge Flutter desktop application designed to revolutionize YouTube content management and analytics for creators, agencies, and businesses. - **Real-Time YouTube Analytics:** Track growth, engagement, and trends instantly

## üì¶ Installation & Setup

1. **Prerequisites:**
   - Linux (Ubuntu recommended)
   - Flutter SDK (>=3.0)
   - Dart SDK
   - mpv & libmpv-dev (`sudo apt-get install libmpv-dev mpv`)
2. **Clone the repository:**

   ```bash
   git clone https://github.com/gabrielle247/batch-tools.git
   cd batch_tech/the_yt
   ```

3. **Install dependencies:**

   ```bash
   flutter pub get
   ```

4. **Run the app:**

   ```bash
   flutter run -d linux
   ```
