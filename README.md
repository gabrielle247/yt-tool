## ü§î Why the_yt?

Streaming YouTube and TikTok content on desktop can be cumbersome, with distractions and limitations. **the_yt** was born out of a need for a focused, privacy-respecting, and efficient tool to:

- Seamlessly stream videos from YouTube and TikTok
- Stay productive and distraction-free
- Avoid violating platform terms of service or copyright
- Batch manage and analyze content for creators and agencies

## üì¶ Installation & Setup

1. **Prerequisites:**
   - Linux (Ubuntu recommended)
   - Flutter SDK (>=3.0)
   - Dart SDK
   - mpv & libmpv-dev (`sudo apt-get install libmpv-dev mpv`)
2. **Clone the repository:**

   ```bash
   git clone https://github.com/gabrielle247/batch-tools.git
   cd batch_tech/the_yt
   ```

3. **Install dependencies:**

   ```bash
   flutter pub get
   ```

4. **Run the app:**

   ```bash
   flutter run -d linux
   ```
